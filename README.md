# Pookie AI — Voice Assistant ✨

A glassy, serverless voice AI companion. Tap the orb, talk, and Pookie talks back — powered by **Google Gemini 2.5 Flash** with your own free AI Studio API key.

## Features
- 🎙️ **Voice-first** — tap-to-talk orb, real-time transcription, spoken replies
- 🧠 Gemini 2.5 Flash via Google AI Studio (BYO key, free tier)
- 🎭 8 personality modes: Girlfriend, Boyfriend, Best Friend, Storyteller (kids' stories), Teacher, Comedian, Motivator, Listener
- 🗣️ Realistic system voices (Google/Microsoft natural voices when available), per-mode pitch/rate
- 💎 Glassmorphism UI, animated orb states (idle / listening / thinking / speaking)
- 🔒 100% client-side — name, age, API key & history stored only in your browser
- 📱 Mobile responsive

## Run
Just open `index.html` in Chrome/Edge (Web Speech API required for mic).
Or host the folder anywhere static (GitHub Pages, Cloudflare Pages, Netlify…).

## API key
Get a free key at https://aistudio.google.com/app/apikey — paste it in onboarding.

## Privacy
For **entertainment only**. No backend. Your messages go directly from your browser to Google's API using your key. Crafted by **zdev**.
