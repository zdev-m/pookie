# Pookie AI ✨

A glassy, serverless AI companion website — powered by **Google AI Studio (Gemini)** with realistic system voices.

## Features
- 🪟 Glassmorphism UI with animated gradient orbs
- 🎭 8 modes: Girlfriend, Boyfriend, Best Friend, Storyteller (kids stories), Teacher, Comedian, Motivator, Listener
- 🔊 Realistic voice replies (browser TTS — picks best Google/Microsoft Natural voices)
- 🎙️ Voice input (Web Speech Recognition)
- 💾 100% serverless — pure HTML/CSS/JS. Hosts anywhere (GitHub Pages, Cloudflare Pages, Netlify, or open `index.html` locally).
- 🔑 BYO API key — get a free one at https://aistudio.google.com/app/apikey

## Run
Just open `index.html` in a modern browser, OR upload all 3 files to Cloudflare Pages / Netlify / GitHub Pages.

## Privacy
Entertainment only. All data (name, age, API key, chat history) stays in your browser's localStorage. Messages go directly from your browser to Google's API using your key.
